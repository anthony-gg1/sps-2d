# SuperMarket Simulator 2D (SPS) 

SuperMarket Simulator 2D é um jogo desenvolvido no **Game Maker Engine** com o objetivo de trazer entretenimento. O projeto tem como inspiração o **SuperMarket Simulator** da Steam e outras obras de jogos de simulação. 

--- 

## Estrutura de Pastas

sps-2d/

├── imgs/                     
├── SuperMarketSimulator2D/  
└── README.md    

---

## Como Baixar e Executar

Clone o repositório:

```bash
git clone https://github.com/anthony-gg1/sps-2d
```

Entre na pasta do projeto:

```bash
cd sps-2d
```

---

Abra o projeto no GameMaker e execute o jogo.
